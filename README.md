## Overview

Arcadia is a desktop Roblox script executor built for Windows users who want a small, direct interface for launching Lua scripts and browsing a bundled script hub. It focuses on fast execution, a broad collection of ready-made scripts, and a clean Windows 10 and Windows 11 workflow that keeps the experience straightforward.

It is presented as a lightweight executor with features aimed at regular use, including auto-update support, batch execution, and a persistent queue stored locally through SQLite. For anyone seeking a no-key Roblox script hub plus executor setup, Arcadia is designed for that purpose.
